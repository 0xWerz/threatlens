# ThreatLens Next Web App

This is the Next.js frontend for ThreatLens.

## Run

```bash
bun run dev
```

## Environment

Copy `.env.example` to `.env.local` and adjust if needed:

```bash
cp .env.example .env.local
```

`NEXT_PUBLIC_THREATLENS_API_URL` defaults to the deployed API:

- `https://threatlens.werz.xyz`

## Build

```bash
bun run build
```
